# 计算机基础 -- 硬件篇
​	首先,经常提起得计算机硬件都有啥?
​	CPU,内存条,影片,显卡,声卡,网卡,主板,机箱电源,键鼠,显示器,音响,摄像头等
​	本次介绍内容为台式机与笔记本电脑的内容混合.

## CPU
```
CPU(中央处理器)包含了运算器和控制器.相当于计算机的"大脑",决定了运算速度的快慢.算是电脑"最重要"的部位
	PC端主流CPU有Intel和AMD两种.
		Intel外号牙膏厂,指的是每次版本更新都是非常些微的性能提升
		AMD近几年(2021)的发展趋势,在性能的释放及版本迭代性能提升都比起Intel亮眼很多.所以有个梗 AMD YES!(表示对AMD产品的认可)
	Intel等级划分(高到低):
		i9 --> i7 --> i5 -->i3 --> 奔腾 --> 赛扬
	AMD等级划分(高到低):
		线程撕裂者--> R9 --> R7 --> R5 --> R3
	购买参考指标:
		主频,缓存,多级缓存,核数,线程数
	常见故障:
		CPU温度过高导致死机,重启.(天气,散热老化,长时间运行,超频) --> 清灰或换散热器
	型号解析:
	i7-10875H --> i7表示的是架构版本
				  10表示的是第十代
				  875表示性能指数
				  H表示标压处理器(性能较强,续航较弱)
	R5-5500U -->  R5表示的是架构版本 
				  5代表第五世代
				  500表示性能指数
			  U代表低压处理器(性能较弱,续航较强)
```



	CPU会分为 好几个架构:
		i3/r3 --> 入门机型用的cpu.性能偏弱
		i5/r5 --> 标准,性能OK
		i7/r7 --> 高性能
		i9/r9 --> 极致性能
	 
	tips:
		1.一般情况下数字越大越牛逼(看整体数字,不是单纯看架构版本.新版本i5/标压的i5性能可能会比上版本/低压i7强).
		2.低压处理器(性能弱,续航强)标压处理器(性能强,续航弱) 
		3.游戏需求+大型软件需求(标压) 日常办公使用(低压)
		4.如果实在不了解信息,可搜索最新版本的"CPU天梯图"里面有性能排行

## 硬盘
​	硬盘主要用作于存储数据.,负责永久性数据存储,速度相较于CPU慢
​	因存储方式不同区分为 
​	机械硬盘(HDD):
​		速度慢,容量大,较便宜
​	固态硬盘(SSD):
​		速度快,容量小,较贵些,适合读多写少
​	购买参考指标:
​		容量,缓存,转速,读写速度,尺寸(笔记本和台式机用的硬盘尺寸是不同的,购买时需要注意),固态接口类型(M.2接口,SATA接口)
​	推荐就是固态+机械 --> 固态作为系统盘,机械做些存储盘
​	计算机里面一些程序的运行速度跟加载速度有关.所以可以把一些常用的软件,游戏放到固态硬盘中

## 内存
​	这里的内存指的是"运行内存",也是用于存储数据,主要存储临时数据,关机后清空,相较硬盘,存储速度更快.
​	内存的好坏也是直接影响了电脑的使用效率,速度.
​	电脑在开机后,会把常用数据从硬盘先读入内存,方便后续CPU使用,提高执行效率.
​	数据读写顺序:
​		CPU --> 内存 --> 硬盘
​	在开发工作时,内存使用率达到80%就需要加内存了.学编程,8G内存打底,频率2666+
​	有时电脑重启后就不卡,是因为释放了垃圾数据(有的应用数据不会随着应用关闭而清空,还会留在内存中)
​		单通道:有一个槽位装内存条
​		双通道:有两个槽位装内存条
​		双通道的运行效率会比单通道更快
​	购买参考指标
​		容量,频率(DDR),尺寸(笔记本和台式机用的内存条尺寸是不同的,购买时需要注意)
​		1.购买时要注意是否和主板相匹配
​		2.加装时,最好购同品牌,同频率产品.避免兼容问题
​		3.购买笔记本时,注意是否支持加装内存条

## 主板
​	主板主要负责将计算机硬件设备直接/间接连接起来,相当于计算机的"骨架".
​	主板的好坏直接决定了计算机运行速度的快慢与稳定.同时主板也可以提供设备接口,以实现拓展功能
​	主板安装了计算机的主要电路系统,一般有:BIOS芯片,I/O芯片,键盘鼠标接口,指示灯插件,拓展插槽,电源插口等.
​	现在主板会根据情况集成3卡.(显卡/网卡/声卡)
​	作为大小分类,有分为大板/小板.一般推荐都是用大板,方便拓展
​	购买参考指标
​		CPU平台,接口数量,支持的技术,规格标准详情

## 显卡
​	显卡有分独显和核显.
​	独显即是一张独立的显卡,功能较为强力.
​	核显(集显)则是显卡与CPU集成,封装在一起.功能比独显弱
​	显卡主要负责于图像数据输出处理.对于大型游戏或者视频渲染,视图,建模生成有比较强的刚需(可用算力进行挖矿)
​	N卡:游戏强项
​	A卡:图像处理
​	购买参考指标
​		显存.位宽,频率
​	由于显卡的体系过于庞大.不是三言两语就能总结完的.要了解显卡可自行搜索显卡天梯图.一般情况下.数字越大,性能越好.
​		RTX3080 > RTX2080 > gtx1080
​	N卡中,同数字情况下,带TI性能更强,带m的则是移动(笔记本电脑)版,性能比起PC版稍弱   gtx1080TI > gtx1080 > gtx1080m
​		N卡的显卡版本分类:
​		GT --> 入门级显卡,近乎淘汰
​		MX --> 低功耗显卡,轻薄本常用
​		GTX --> 目前的主流,中高端显卡
​		RTX --> 目前的高端显卡
​	TI --> 加强版  super --> 性能在标准版与TI之间的强化版
​	tips:
​		同版本的显卡,在功率上也会有不同,会影响性能表现rtx3060(80W,95W,130W)

## 操作系统
​	一台完整的计算机,是由硬件和软件组成的,硬件先告一段落,现在了解软件.先说操作系统.
​	操作系统主要负责管理硬件,驱动硬件,以及软件应用程序平台.操作系统本身就是软件,可以理解为是一个管家.管理电脑的事务.
​	一般的PC操作系统:Linux,windows,MAC (UNIX偏小众)
​		Linux:RedHat,Ubuntu,debian,CentOS等
​			Linux的系统发展是有好几个不同的分支而不是一条线发展下去的,因此要学习使用Linux系统,先了解自己要做的是什么类型再选择相应系统.Linux相对上手难度高,主要是由程序员,工程师使用.
​		windows:XP,WIN7,WIN8,WIN10
​			windows相对普及化,适合大部分普通人使用.大部分使用的第一系统
​		MAC:
​			本人不用苹果系列,不了解就不BB了,欲知详情可自行百度.
​	一般情况下都是一机一系统,而需要用到多个系统的时候,一般就是
​		1.双系统电脑
​		2.虚拟机软件实现
​	两者等后续正式使用Linux再具体补充
​		而系统又可以再分为个人用以及服务器用(网络版).
​	个人用:windows系统为主,主要是各种图形界面便于日常使用
​	服务器用:Linux系为主,主要用于服务器的部署,开发,运维.图形操作较少甚至没有
